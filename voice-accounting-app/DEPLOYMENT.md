# 语音记账应用部署说明

## 应用状态
✅ 应用已成功部署并运行在: https://192.168.2.28:3000

## 已修复的问题
1. **客户创建失败** - 修复了手动创建客户无法保存的问题
2. **发票编辑无法更换客户** - 修复了编辑发票时无法更改客户的bug
3. **语音识别优化** - 增强了中文姓名识别，支持"张三购买了"、"李四要了"等模式

## 管理命令
```bash
# 启动应用
./start.sh

# 停止应用
./manage.sh stop

# 重启应用
./manage.sh restart

# 查看状态
./manage.sh status

# 开启自动重启监控（可选）
./monitor.sh
```

## 项目结构
```
voice-accounting-app/
├── src/              # 源代码
├── prisma/           # 数据库配置
├── public/           # 静态资源（已清理测试文件）
├── certs/            # SSL证书
├── manage.sh         # 管理脚本
├── monitor.sh        # 监控脚本
├── start.sh          # 启动脚本
└── server.js         # 服务器配置
```

## 注意事项
1. 应用使用HTTPS运行，这是浏览器语音识别功能的要求
2. 使用nohup保持应用在后台持续运行
3. 所有测试文件和开发文档已清理
4. 应用会在系统重启后需要手动启动（或使用monitor.sh自动监控）
